# Description
A dashboard created using [Buefy](https://buefy.github.io), [Bulma](https://bulma.io/) and [Vue](https://vuejs.org/).

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Lints and fixes files
```
yarn run lint
```

### Run your unit tests
```
yarn run test:unit
```
